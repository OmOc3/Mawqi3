# Mawqi3 Field — تطبيق موقعي الميداني

تطبيق Expo مرافق لنظام Mawqi3. الهدف منه فتح مسار الفني بسرعة من الهاتف: شاشة مسح، إدخال رقم المحطة يدويًا عند الحاجة، وفتح نموذج التقرير الآمن على بوابة الويب.

## التشغيل

```bash
npm install
npm run start
npm run web
```

انسخ `.env.example` إلى `.env` واضبط:

```bash
EXPO_PUBLIC_MAWQI3_WEB_BASE_URL=https://your-mawqi3-domain.com
```

## الملاحظات الأمنية

- لا توجد مفاتيح Firebase Admin أو Gemini داخل تطبيق الموبايل.
- التطبيق يفتح روابط Mawqi3 المحمية على الويب، والصلاحيات تظل محكومة بالجلسة والدور.
- الوضع الداكن مدعوم من داخل التطبيق: النظام، فاتح، داكن.
